# 图片资源说明

## 需要的图片文件

请将以下图片文件放置在此目录中：

### 1. 图标类
- `empty.png` - 空状态图标（建议尺寸：200x200px）
- `error.png` - 错误状态图标（建议尺寸：200x200px）
- `logo.png` - 应用图标（建议尺寸：200x200px）

### 2. 占位图
- `placeholder.png` - 默认占位图（建议尺寸：400x300px）

## 图片规格要求

- 格式：PNG（推荐）或 JPG
- 大小：建议单个图片不超过 100KB
- 分辨率：适配不同屏幕密度

## 替代方案

如果暂时没有设计资源，可以使用以下在线工具生成：
- [Iconfont](https://www.iconfont.cn/) - 阿里巴巴矢量图标库
- [Unsplash](https://unsplash.com/) - 免费高质量图片
- [Flaticon](https://www.flaticon.com/) - 免费图标

## 注意事项

1. 确保图片文件命名与代码中引用的路径一致
2. 建议对图片进行压缩优化，减少小程序包体积
3. 遵循微信小程序图片规范，避免使用过大的图片文件
