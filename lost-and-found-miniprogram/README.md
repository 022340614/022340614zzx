# 校园失物招领小程序

一个基于微信小程序云开发的校园失物招领平台，帮助师生更方便地发布和查找失物信息。

## 功能特性

### 核心功能
- 🔍 **信息发布**：支持发布招领和寻物信息
- 📱 **图片上传**：最多支持上传3张物品图片
- 🔎 **智能搜索**：支持按物品名称、地点搜索
- 🏷️ **分类筛选**：按招领/寻物类型筛选
- 📞 **安全联系**：联系方式脱敏显示，保护隐私
- 👤 **个人中心**：管理自己发布的信息

### 技术特性
- ☁️ **云开发**：基于微信云开发，无需自建服务器
- 📱 **响应式设计**：适配不同尺寸的移动设备
- ⚡ **性能优化**：图片懒加载、数据缓存
- 🔒 **安全可靠**：数据验证、权限控制

## 快速开始

### 1. 环境准备
- 安装微信开发者工具
- 注册微信小程序账号
- 开通云开发服务

### 2. 项目配置
1. 在微信开发者工具中导入项目
2. 在 `project.config.json` 中配置你的 AppID
3. 初始化云环境

### 3. 数据库初始化
1. 上传并部署云函数 `init-db`
2. 调用云函数初始化数据库结构

### 4. 运行项目
1. 编译项目
2. 在开发者工具中预览

## 配置说明

### 学校配置
在 `app.js` 的 `schoolConfig` 中配置学校相关信息：

```javascript

  locations: [
```

## 开发指南

### 添加新页面
1. 在 `pages` 目录下创建页面文件夹
2. 在 `app.json` 的 `pages` 数组中添加页面路径
3. 创建对应的 `.wxml`, `.wxss`, `.js`, `.json` 文件

### 云函数开发
1. 在 `cloudfunctions` 目录下创建云函数文件夹
2. 编写 `index.js` 和 `package.json`
3. 上传并部署云函数

## 常见问题

### Q: 图片上传失败怎么办？
A: 检查云存储权限和文件大小限制（建议不超过2MB）

### Q: 搜索功能不生效？
A: 确保数据库已创建文本索引

### Q: 联系方式显示异常？
A: 检查联系方式格式，支持手机号、邮箱、微信号
