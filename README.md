# 🏫 校园失物招领智能PWA平台

## 🌟 项目介绍

**湖北民族大学校园失物招领智能平台**是一个基于现代Web技术的渐进式网页应用(PWA)，专为校园环境设计，提供智能化的失物招领服务。

### 🎯 核心特色

- **🏫 湖北民族大学专属** - 集成校园地图和地点信息
- **📱 PWA技术** - 可安装到手机主屏幕，支持离线使用
- **🤖 智能功能** - AI识别、实时聊天、数据分析等高级功能
- **🎨 现代设计** - 响应式界面，完美适配各设备
- **⚡ 快速部署** - 一键部署到GitHub Pages

## 🎯 使用指南

### 发布物品
1. 点击"发布"标签页
2. 选择信息类型（寻物/招领）
3. 填写物品信息和校园地点
4. 可使用地图选择具体位置
5. 上传物品图片（可选）
6. 提交发布

### 搜索物品
1. 在搜索框输入关键词
2. 可使用分类筛选
3. 支持校园地点搜索
4. 查看匹配结果

### 使用校园地图
1. 点击导航栏"校园地图"按钮
2. 浏览校园各区域
3. 点击地点查看详情
4. 使用导航功能
